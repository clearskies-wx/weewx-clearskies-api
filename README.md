# weewx-clearskies-api

Phase 1 placeholder — real content lands during Phase 2 of the Clear Skies project.

HTTP/JSON API serving (a) read-only access to the [weewx](https://github.com/weewx/weewx) archive database, and (b) external provider data (forecast / AQI / alerts / earthquakes / radar) via per-provider plugin modules internal to this repo.

Part of [Clear Skies](https://github.com/inguy24/weewx-clearskies-stack), a modular modern weather UI for weewx.
